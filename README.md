# Sales & Warehouse Operations Automation System

## Project Overview

This project is an end-to-end business operations automation and analytics solution built using CSV datasets, MySQL, Python, and Power BI.

The system automates inventory management, shipment monitoring, KPI reporting, and business analytics for a beverage/syrup distribution company.

The project simulates a real-world workflow where sales transactions are processed, inventory is updated automatically, operational alerts are generated, and management dashboards provide actionable insights.

---

## Business Problem

Manual inventory tracking, shipment monitoring, and KPI reporting are time-consuming and prone to errors.

This project automates:

* Inventory stock updates
* Low-stock detection
* Out-of-stock alerts
* Shipment monitoring
* Delayed shipment reporting
* Executive KPI reporting
* Sales performance analytics
